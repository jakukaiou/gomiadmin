# gomiadmin

## usage
1. cd ~
1. git clone https://github.com/jakukaiou/gomiadmin
1. cd gomiadmin
1. mkdir gomipic
1. (add gomifile in gomipic directory)

### put gomi in Desktop
~~~
sh ~/gomiadmin/gomigen.sh
~~~

### throw gomi in Desktop
~~~
sh ~/gomiadmin/gomithrow.sh
~~~

### delete all gomi in Trash
~~~
sh ~/gomiadmin/gomidel.sh
~~~
